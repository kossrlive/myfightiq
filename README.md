# MyFightIQ

A 3D browser-based fighting game built with Three.js and Socket.IO.

## Features

- Single-player and multiplayer modes
- Real-time 3D combat with various attack moves
- Dynamic animations and physics
- Multiplayer support with rooms and player names
- Background music and sound effects
- Responsive controls with keyboard input
- Health system with blocking mechanics
- Victory/defeat animations

## Controls

- WASD: Move character
- Arrow Keys: Attack moves
- F: Ultra hit
- Space: Jump
- Shift: Block

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Open `http://localhost:8080` in your browser to play

## Development

The project uses the following technologies:
- Three.js for 3D graphics
- Socket.IO for multiplayer functionality
- Node.js for the server
